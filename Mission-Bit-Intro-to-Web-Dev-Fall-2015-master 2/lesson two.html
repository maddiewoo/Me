What is HTML?

HTML (Hypertext Markup Language) is the basic skeleton of the internet. At it's simplest form, like this website, it looks incredibly boring. But if the basic HTML structure didn't exist, there would be no websites at all.

HTML is a language written in and between tags that look like this:

<NAME OF THE TAG GOES HERE>

Each tag, once opened, must be closed, because the browser interprets within the tags--if you don't close a tag like this:

</NAME OF THE TAG GOES HERE>

It will continue interpreting the meaning of the tag for all the code beneath it.

Every tag has a different meaning. The <p> tag, for instance, denotes the opening and closing of a paragraph.

Tags can be nested within each other so that both tags affect the things sandwiched between them. For instance, the <strong> tag is used to make things bold and the <em> tag is used to make things italicized, but you can put them together like this:

<strong>
    <em>
        YOUR TEXT GOES HERE
    </em>
</strong>

To make things both bold and italicized.
